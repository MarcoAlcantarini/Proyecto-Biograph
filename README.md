Este proyecto (BioGraph) trata de un análisis que modela una red de interacción Proteína-Proteína (PPIN), como un grafo no dirigido, permitiendo identificar proteínas esenciales, rutas metabólicas para el descubrimiento de fármacos y calcular las dianas terapéuticas más relevantes.
